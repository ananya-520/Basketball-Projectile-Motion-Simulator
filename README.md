# Basketball Projectile Motion Simulator

A MATLAB-based projectile motion simulator that calculates the optimal launch angle and initial velocity required for a basketball to clear an obstacle and land in a hoop. The project applies classical equations of motion to simulate trajectories, compute time of flight and maximum height, and visualize the projectile path. It also includes an interactive MATLAB GUI where users can input obstacle dimensions, explore valid launch angles, animate the trajectory, and inspect the projectile's position and velocity at any point in time.
